🚀 VMess / VLESS / Trojan / SS Tunnel (Recommended)

VMess/VLESS/Trojan/Shadowsocks link save করলে server-এ Xray-core auto-start হবে এবং local SOCKS5 proxy (socks5://127.0.0.1:10808) হিসেবে yt-dlp-এ inject হবে। Direct socks5://... link দিলে Xray skip হয়ে সরাসরি use হবে।

🌐 Residential Proxy (alternative to VMess)

Webshare/SmartProxy/BrightData থেকে credentials নিয়ে এখানে paste করুন।

🔑 Google Drive OAuth Credentials

Google Cloud Console → OAuth 2.0 Client IDs → Web application থেকে নিন। Redirect URI অবশ্যই app-এর domain হতে হবে।

🔥 Sub Burner — Upload Credentials

Sub Burner tab-এর upload targets-এর জন্য credentials এখানে save করুন।

💡 YouTube Access Token: Google OAuth দিয়ে login করার পর token পাওয়া যায়। শুধু private video upload হবে।
