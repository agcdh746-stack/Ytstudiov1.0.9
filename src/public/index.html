⚠️ Personal use only. এই page থেকে দেওয়া secrets/keys server এর persistent disk-এ save হয়। Public URL শেয়ার করবেন না।

🚀 VMess / VLESS / Trojan / SS Tunnel (Recommended)

VMess/VLESS/Trojan/Shadowsocks link save করলে server-এ Xray-core auto-start হবে এবং local SOCKS5 proxy (socks5://127.0.0.1:10808) হিসেবে yt-dlp-এ inject হবে। Direct socks5://... link দিলে Xray skip হয়ে সরাসরি use হবে।

🌐 Residential Proxy (alternative to VMess)

Webshare/SmartProxy/BrightData থেকে credentials নিয়ে এখানে paste করুন।

🔑 Google Drive OAuth Credentials

Google Cloud Console → OAuth 2.0 Client IDs → Web application থেকে নিন। Redirect URI অবশ্যই app-এর domain হতে হবে।

🔥 Sub Burner — Upload Credentials

Sub Burner tab-এর upload targets-এর জন্য credentials এখানে save করুন।

💡 YouTube Access Token: Google OAuth দিয়ে login করার পর token পাওয়া যায়। শুধু private video upload হবে।

📨 Telegram Upload — Channel Credentials

Waz Clipper / নাটক Clipper / Bulk Downloader-এর "Telegram Upload" সেকশনের জন্য। একটাই bot token, প্রতিটা channel-এর নিজের chat ID।

🚀 50MB+ ফাইল সরাসরি video হিসেবে পাঠাতে (document না) — নিচের ২টা ফিল্ড দাও। my.telegram.org/apps থেকে ফ্রি-তে পাওয়া যায় (phone number verify করতে হয়)। এগুলো দিলে এই app নিজেই (কোনো আলাদা service ছাড়া) Telegram-এর সাথে সরাসরি কথা বলে বড় ফাইল আপলোড করবে, ~2000MB পর্যন্ত।

⚙️ Advanced: Local Bot API Server (এটার বদলে)

নিজেই একটা telegram-bot-api server চালালে (Railway-এ আলাদা service হিসেবে), তার URL এখানে দিতে পারো। এটা দেওয়া থাকলে এটাই ব্যবহার হবে, উপরের API ID/Hash না।

💡 যেসব channel-এর chat ID খালি থাকবে, সেই channel-এ upload করতে গেলে error দেখাবে — বাকিগুলো ঠিকমতো কাজ করবে। API ID/Hash না দিলে 50MB-এর বেশি ফাইল document হিসেবে যাবে (চলবে, কিন্তু video preview থাকবে না)।
